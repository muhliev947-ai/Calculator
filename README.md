🧮 Calculator App

Минималистичный и быстрый калькулятор, созданный на JavaScript и Tailwind CSS.  
Проект демонстрирует работу с модульной архитектурой, обработкой событий и кастомным UI.
✨ Возможности

    ➕ Базовые операции: сложение, вычитание, умножение, деление

    🔄 Очистка и сброс значений

    ⌨️ Поддержка ввода с клавиатуры

    🎨 Адаптивный интерфейс на Tailwind CSS

    ⚙️ Модульная структура JS

🛠️ Технологии

    JavaScript (ES6+)

    Tailwind CSS

    HTML

    Инструменты: Git, VS Code, Linux

🎯 Цели проекта

    Практика модульной архитектуры в JavaScript

    Работа с DOM и обработкой событий

    Создание адаптивного UI с Tailwind

    Реализация логики вычислений без сторонних библиотек
